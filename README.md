## Getting Started with Node

- First, install:

```bash
npm run install
```

- then build:

```bash
npm run build
```

- finally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Getting Started with Bun

```bash
  bun install
  bun dev
```
